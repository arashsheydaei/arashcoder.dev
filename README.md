### Installation

1. Clone the repository:
```bash

```

```bash
npm install

```

3. Create a `.env` file in the root directory and add your Firebase configuration:
```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
```

```bash
npm run dev
# or
yarn dev
```
